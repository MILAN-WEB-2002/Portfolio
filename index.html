Main Project

REACT: AI-Powered Litter Detection System

React – The Fourth R in Waste Management is an AI-powered system that detects illegal waste disposal in real time using a Jetson Nano–camera unit. It identifies pedestrians, two-wheelers, and cars discarding waste, captures a 5-second video with metadata like timestamp and license plate, and uploads the evidence to the cloud for enforcement. Supported by the Alappuzha Municipality and Steag, React introduces a new “R”—React—to complement Reduce, Reuse, and Recycle, enabling smarter and cleaner cities.

Key Features:

  • Real-time detection of littering actions using YOLOv8 and pose estimation
  • Automatic classification of offenders: pedestrians, two-wheelers, and cars
  • 5-second video evidence capture with timestamp and location
  • License plate recognition using EasyOCR for offender identification
  • Cloud-based storage and admin dashboard for review and enforcement

Technologies Used:

Python OpenCV YOLOv8 Pose Estimation EasyOCR Jetson Nano Firebase / Cloud React.js (Admin Dashboard)
